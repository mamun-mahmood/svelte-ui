## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/mamun-mahmood/svelte-ui.git
   cd svelte-ui
   ```

2. Install dependencies:

   ```bash
   npm install
   "if there's dependency conflict run"
   npm install -f
   ```

3. Run the application:

   ```bash
   npm run dev
   ```
